What is SkyoClient?

SkyoClient is the do it all client that has an advanced API to create unique visual effects for supported servers, for course SkyoBlock will use the SkyoClient API to add new HUD elements so you can gain more knowledge about the world around you. You can set it up on your server to support the effects or require the effects, due to the nature of SkyoBlock, it will support but not require SkyoClient.
